## 🚀 Pós-publicação

Depois de publicar:

1. **Stories**: faz 2-3 stories a apontar para o post novo
2. **Resposta**: passa 2h pelos comentários — responde a tudo
3. **DM**: se houver pais a perguntar coisas específicas, responde
4. **Cross-post**: Facebook (mesmo carrocel funciona) + LinkedIn (mais formal)
5. **Reddit**: partilha em r/portugal se sentires que cabe (sem auto-promo agressivo)

## ❤️ Se algo correr mal no Gemini

Gemini Nano Banana às vezes:
- **Distorce texto** → re-tenta com instrução "use clear, readable serif/sans font"
- **Inventa palavras** → revê SLIDE A SLIDE, não confies no batch
- **Muda cores** → especifica HEX em CAPS em cada prompt
- **Faz infantilizado** → adiciona "professional, NOT childish, suitable for adults" no prompt

Se o batch master falhar, usa os prompts individuais. Pelo menos um deles vai resultar.
