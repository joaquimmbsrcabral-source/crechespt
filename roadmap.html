Em exploração

Ideias que estamos a estudar. Sem garantia de chegarem — depende de fazerem sentido.
  • 👥
    Conta para creches com auto-edição

    Dashboard onde cada creche pode actualizar os próprios dados — mensalidades, horários, fotos, vagas — sem ter de passar pelo admin.

  • 📋
    Lista de espera tracker

    Os pais acompanham em que posição estão em cada creche onde se inscreveram. Alertas quando avançam.

  • ⚖️
    Comparador lado a lado

    Comparar 3 creches: mensalidades, idades aceites, horário, distância, equipa, número de crianças por sala.

  • 🔔
    Notificações de novas creches ou vagas

    Quando aparecer uma creche nova no teu concelho, ou uma vaga numa creche que tens guardada, ser avisado.

  • 🔌
    API pública read-only

    Disponibilizar o dataset agregado para investigadores, jornalistas e developers que queiram fazer análises ou ferramentas próprias.

  • 🗓
    Agendamento de visitas

    Marcar visita a uma creche directamente na app, com sincronização para o calendário.

  • 💰
    Médias de mensalidades por concelho

    Dados agregados (e anónimos) de mensalidades reportadas por pais, para teres uma noção realista antes de telefonar.

Provavelmente nunca

Ideias que não estão na nossa direcção actual. Não é compromisso — apenas o que faz sentido hoje.
  • 🚫
    Publicidade na app

    Banners, links patrocinados, ou creches que pagam para subir no ranking. Hoje a Creches.app é totalmente sem ads — e essa é a direcção que faz sentido continuar.

  • 🔒
    Login obrigatório para ver o mapa

    O mapa é visível sem registo. Login só existe para sincronizar a tua pesquisa entre dispositivos — e é opcional.

  • 💸
    Venda de dados a terceiros

    Os dados dos utilizadores não saem da Creches.app. Não há programa de afiliados disfarçado nem partilha com "parceiros".

Porque é que isto está aqui?

Porque os pais merecem saber o que estamos a fazer com a Creches.app — não só o que já existe, mas também o que está a vir. E porque é justo dizer também o que não estamos a planear.

Esta página é actualizada quando há novidades. Os itens em "exploração" podem nunca chegar, ou podem chegar mais cedo do que esperado — é nesse compromisso flexível que esta página vive.
